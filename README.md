# 프론트엔드 면접 준비

알고리즘 문제 풀이, CS 지식, 코딩 과제를 포함한 프론트엔드 면접 준비 자료입니다.

## 디렉토리 구조

### 📁 algorithms/

- 알고리즘 문제 풀이 및 설명
- 자료구조와 알고리즘 유형별 정리
- 리트코드 솔루션 포함

### 📁 frontend-cs/

- 웹 및 프론트엔드 CS 지식
- 브라우저 렌더링, 네트워킹, React 등
- 성능 최적화 및 디자인 패턴

### 📁 coding-challenges/

- 실전 코딩 과제 연습
- CRUD 구현
- 인증 구현
- React 컴포넌트 설계
- 성능 최적화 연습

## 사용 방법

1. 학습 목적에 따라 디렉토리 탐색
2. 각 섹션별 상세 설명은 해당 README 참고
3. 코드 솔루션에는 주석과 설명 포함
